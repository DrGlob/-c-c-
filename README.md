# Тестовое задание в компанию «Инфотекс»

## Задание 1
Есть микросхема EEPROM памяти с интерфейсом SPI (например, `25LC040A`).
Есть драйвер, позволяющий через переключение битов (bit-banging) управлять линиями микросхемы.
Необходимо реализовать С++ класс (набор классов) для работы с микросхемой посредством драйвера.

Требуется запись данных по адресу — бита/байта/массива байт. То же самое с чтением.
Что изменится, если будет необходимо реализовать драйвер для работы с NOR памятью (например, `W25Q128`).

Все методы, константы, данные, перечисления и т.п. должны быть документированы.
Предпочтение — формат Doxygen.

## Задание 2
Требуется реализовать кастомный аллокатор на языке С (не С++).
Необходимо реализовать функции `malloc` и `free`.
Запрашивается выделение памяти только двух размеров — 15 байт и 180 байт.

Обратить внимание на быстродействие и эффективность работы с памятью.

Код должен работать на 8/16/32-битной архитектуре.
